                         │        Input (User / JSON)     │
                         └────────────────────────────────┘
                                       │
                                       ▼
                         ┌────────────────────────────────┐

                         └────────────────────────────────┘
                                       │
                                       ▼
                         ┌────────────────────────────────┐
                         │        Mode Selector           │
                         │  (invoice / math / policy…)    │
                         └────────────────────────────────┘
                                       │
                                       ▼
                         ┌────────────────────────────────┐
                         │        Skill Selector          │

                         │   skill module by schema/tags  │
                         └────────────────────────────────┘
                                       │
                                       ▼
                     ┌────────────────────────────────────────┐
                     │               Skill                    │
                     │  def metadata(): …                     │
                     │  def execute(input): output            │
                     └────────────────────────────────────────┘
                                       │
                                       ▼
                         ┌────────────────────────────────┐
                         │          Output                │
                         │  (typed dict)                  │
                         └────────────────────────────────┘
                                       │
                                       ▼
                         ┌────────────────────────────────┐
                         │        Logging Layer           │
                         │  - INPUT → MODE → SKILL → OUT  │

                         ┌────────────────────────────────┐
